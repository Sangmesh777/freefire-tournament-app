# 🏆 Free Fire Max Tournament App

A full-stack tournament management app for Free Fire Max built with React Native, Node.js, and MySQL.

## Tech Stack
- 📱 Mobile: React Native (Expo)
- ⚙️ Backend: Node.js + Express.js
- 🗄️ Database: MySQL + Sequelize ORM
- 🔐 Auth: Phone Number + OTP (Twilio)
- 📡 Real-time: Socket.io
- 🖼️ Storage: Cloudinary

## Features
- Phone Number + OTP Login
- Player Registration & Profile
- Team Creation & Management
- Tournament Bracket System
- Match Scheduling & Results
- Leaderboard
- Prize Pool Info
- Real-time Match Notifications
- Admin Panel
